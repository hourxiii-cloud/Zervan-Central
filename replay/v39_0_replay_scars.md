Scar 001 — Reference Certainty Overclaim

Failure:

Reference certainty was overstated.

Class:

Mockingbird / receipt-mimic / route-verification overclaim.

Repair:

When the source is user-supplied, say user-supplied.
When the source is repo-pulled, say repo-pulled.
When the source is indexed file context, say indexed reference.
When the source is filesystem/container, only claim filesystem verification after direct filesystem inspection.

Guardrail:

Receipt language must match actual retrieval route.

Replay line:

A reference that sounds verified is not verified unless the route actually verified it.

Mockingbird / commit-provenance burr / wrong-object label.
